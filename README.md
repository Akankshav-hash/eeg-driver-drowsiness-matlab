# EEG-Based Driver Drowsiness Detection Using MATLAB and SVM

## Overview

This project presents an EEG-based driver drowsiness detection system that combines signal processing and machine learning techniques in MATLAB. EEG signals are preprocessed, relevant features are extracted, and a Support Vector Machine (SVM) classifier is used to identify whether a driver is alert or drowsy.

## Features

* EEG preprocessing
* Butterworth band-pass filtering
* Feature extraction from EEG bands
* SVM-based classification
* MATLAB GUI for visualization
* Real-time drowsiness indication

## Project Structure

* `code/` – MATLAB scripts
* `docs/` – Flowcharts, GUI images, confusion matrix
* `report/` – Project report

## Tools Used

* MATLAB
* Signal Processing Toolbox
* Machine Learning techniques (SVM)

## How to Run

1. Open the MATLAB project.
2. Load the EEG dataset using the buildDataset.m
3. Execute preprocessing and feature extraction scripts.
4. Train or load the SVM model.
5. Launch the GUI and test classification.

## Results

The system successfully distinguishes between alert and drowsy EEG patterns and provides an interactive interface for monitoring classification outcomes.
